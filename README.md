# High-Dimensional-Humanoid-Locomotion-control-using-PPO
PPO-based Deep RL framework for stable 3D humanoid locomotion in MuJoCo Humanoid-v4. Agent controls 17 joint DoFs, learning energy-efficient bipedal walking from reward signals alone. 12 parallel workers, 15M timesteps, Stable-Baselines3 + PyTorch.
